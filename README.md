# MSCI — MapleStory Class Indicator

MSCI is a MapleStory Classic World / nostalgic MapleStory class personality test.

It uses a dimension-based scoring model instead of direct A/B/C/D class voting:

1. User answers questions.
2. Answers update hidden playstyle dimensions.
3. Dimensions are matched against first-job class profiles.
4. The selected first job unlocks a second-job branch test.
5. The result page shows main class, second-job branch, match confidence, secondary personality, and explanation.

## Planned model

- First job: Warrior, Magician, Thief, Archer
- Second job: Fighter, Page, Spearman, Ice/Lightning Wizard, Fire/Poison Wizard, Cleric, Assassin, Bandit, Hunter, Crossbowman
