# Personality images

Put final personality images in this folder and name them by the four-letter code.

Recommended PNG filenames:

- `SLAY.png` — 剑客 / 正统砍王
- `SHLD.png` — 准骑士 / 盾牌信仰人
- `POLE.png` — 枪战士 / 长枪后期人
- `ZAPZ.png` — 冰雷法师 / 冰雷控图人
- `TOXI.png` — 火毒法师 / 毒火研究人
- `HEAL.png` — 牧师 / 队伍亲妈人
- `STAR.png` — 刺客 / 标飞装帅人
- `STAB.png` — 侠客 / 匕首背刺人
- `KITE.png` — 猎人 / 风筝猎人
- `SNIP.png` — 弩弓手 / 冷门狙击人

The app will automatically load:

```text
/MSCI/personalities/CODE.png
```

Example:

```text
/MSCI/personalities/STAR.png
```

If the image file does not exist, the app shows a placeholder card instead.
